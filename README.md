# mlokokulon-web

Website profil Kelurahan Mlokomanis Kulon, Wonogiri — proyek KKN. Lihat [CLAUDE.md](CLAUDE.md) untuk ringkasan spesifikasi dan [PRD lengkap](PRD_Web_Profil_Desa_Mlokomanis_Kulon_v8%20(1).md).

## Getting Started

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Stack

Next.js (App Router) + TypeScript + Tailwind CSS, hosting Vercel, backend Firebase (Firestore, Auth, Storage).
